# Project Instructions

1. **No Premium Features**: The application must remain entirely free and fully functional. Do not introduce any premium tiers, subscription dialogs, paywalls, or locked "Pro" features.
2. **No AI or Paid APIs**: Do not integrate the Gemini API or any other third-party cloud APIs that require payment or developer API keys. 
3. **Local/Offline First**: Core functionality must rely on local processes. Use local database (Room) lookups, local keyword matching, and offline logic rather than offloading to external services.
